#  Tic Tac Toe — Advanced Tkinter Edition

A visually appealing and fully functional **Tic Tac Toe game** built using **Python’s Tkinter GUI library**.  
This project includes an intelligent **AI opponent** (Minimax algorithm), difficulty modes, a responsive fullscreen interface, and smooth animations — designed to feel modern and polished.


##  Features

✅ **Single-player and Two-player modes**  
Play against a smart AI (with Easy or Hard mode) or challenge a friend locally.

✅ **AI with Minimax Algorithm**  
The AI makes optimal moves in *Hard mode* using recursion and game-tree evaluation.

✅ **Beautiful Dark UI**  
Custom colors, gradients, and animations for a professional look.

✅ **Scoreboard System**  
Tracks wins, losses, and ties across multiple rounds.

✅ **Fullscreen Responsive Design**  
Automatically adjusts to any screen size; press `Esc` to exit fullscreen.

✅ **Restart & Difficulty Controls**  
Easily restart the game or switch between difficulty levels in one click.



### 1. Requirements

- **Python 3.8+**
- **Tkinter** (comes pre-installed with most Python distributions)
- **No external dependencies!**
